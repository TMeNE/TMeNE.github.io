自我介绍

三种类型

​ 注意：介绍用正向的词，不能负向 要有升华

  • 轻松愉悦式
  • 诙谐幽默式
  • 儒雅式

时间分类

  • 一分钟自我介绍

    基础介绍 初级框架

    思考自我介绍的目的是什么（避免跑题）

    我的单位 来自哪里

  • 三分钟自我介绍

    有内容、厚度 性格特点等 企业的介绍 包括哪个产品的介绍

    通过哪个产品反过来又让大家记住我 与相关的资料贯穿起来

    如：我是个吃货，所以我就进入了餐饮行业。

    我是XX公司的，我们有一道非常经典的菜，XX样的菜，欢迎XX，我请你品尝。

  • 五分钟自我介绍

    内容扩展 信念价值观（你是个很立体的人）

    如：我喜欢下棋、我喜欢旅游、在座想旅游的时候可以咨询我、我为什么喜欢旅游（生活在祖国大好河山下我能感受到什么）、[再扩展一下]我去年刚去过西藏，看到的风景对我的影响

开场方式

  • 赞美式（容易和大家拉近距离）

    如：爽爽贵阳、为我们爱学习的同学鼓个掌，今天天气这么热

  • 引用时（上来就拉近距离）

    用一些名人名言、刚才发生的故事、共同的经历、当今流行的话题、一些敏感的事件

  • 讲故事

    因主题而定 笑话、故事 收集相关资料

  • 提问式（进行互动）

    提问引导 自我介绍时也可以提问

  • 自嘲幽默式

    自黑

  • 场景引导

    即景生情式

开场万能公式

​ 顺序可以调整

  • 问好
  • 感谢
  • 自我介绍
  • 经典开场
  • 主题（告诉大家 便于大家理解 当演讲比较短时放到最后-画龙点睛-但有风险-逻辑性）
  • 对对方的好处（思考受众想得到的是什么）

心理障碍

  • 自卑
  • 准备不充分
  • 过于在意别人的看法
  • 追求完美
  • 害怕出丑

十大信念

​ 心理暗示 信念植入

  • 我喜欢讲话
  • 我喜欢我的声音
  • 我喜欢我自己（喜欢自己的十大理由 十条–>二十条–>三十条）
  • 我喜欢这个舞台
  • 只要有上台的机会 我就会冲上舞台
  • 只要有开口说话的机会我就会开口说话
  • 只要开口说话我就能用肢体语言表达
  • 只要有机会说话我就愿意用眼神交流
  • 只要有学习的机会我就会跟别人学习
  • 我相信我的口才一定可以练好

表达方法

  • 自问自答式

  • 停顿式（停顿处为重点 相当于加粗飘红）

  • 举例子

  • 音调

  • 语速（正常语速：120-150字一分钟 比平时要慢 娓娓道来 沟通的效果取决于对方的回应）

    口部操训练

    发音方法

    • 口腔共鸣
    • 胸腔共鸣法
    • 腹式呼吸法
    • 鼻腔共鸣法

    练习语句：让我们进入一个梦幻的世界
